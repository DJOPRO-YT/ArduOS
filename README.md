# ArduOS
- ArduOS is the Official Operating System for the ArduConsole. Designed to be lightweight so it can run on the ATMEGA328 MCU with only 2KB of RAM.

## Features: 
- Optimized for the ATMEGA328 MCU.
- Graphics Support for the LCD Character Display
- Cartridges Support for ArduConsoles Games
- And much more !
